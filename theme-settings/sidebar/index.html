整体样式

默认情况下，侧边栏仅在文章页面（拥有目录列表时）才显示，并置于左侧位置。你可以在 主题配置文件 中的sidebar选项中修改它。

侧边栏有几个选项，包括position、width、display、padding和offset。

  • position
  • width
  • display
  • padding
  • offset

通过修改sidebar.position的值，可以设置侧边栏的位置，该值可以是以下之一：

  • left → 放置在屏幕左边。
  • right → 放置在屏幕右边。

以像素为单位来修改sidebar.width的值，可以改变侧边栏的宽度，例如：

通过修改sidebar.display的值，可以设置侧边栏显示的条件，可以是以下之一：

  • post → 仅在有索引的文章中显示侧边栏。
  • always → 在所有页面中显示侧边栏。
  • hide → 在所有页面中隐藏侧边栏（但可以由用户手动打开）。
  • remove → 完全删除侧边栏。
