  • THEME-SETTINGS
  • CUSTOM-PAGES

  • 设置 Front-matter →
  • 编辑菜单

添加「标签」页面

新建「标签」页面，并在菜单中显示「标签」链接。「标签」页面将展示网站的所有标签，如果文章没有标签，此页面将为空。

  • 添加新页面 →
  • 设置页面类型 →
  • 编辑菜单

添加「分类」页面

添加「分类」页面与添加「标签」页面方法一样，只是名称不同。只需将 tags 替换成 categories 即可。
